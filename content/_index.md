# Welcome to the Houston Linux User Group

Here at HLUG we create a safe place for all Linux activity in Houston. Whether
you are looking to socialize with Linux loving peers, promote the adoption of
Linux in a particular venue, or learn about Linux and its software offerings,
the Houston Linux User's Group can assist you in achieving your goals, with
Linux!
